# AVO-class-automation
AVO class automation
Multi-Task BiLSTM for Automated Seismic AVO Inversion and ClassificationThis repository contains a Deep Learning implementation using a Bidirectional Long Short-Term Memory (BiLSTM) architecture to perform simultaneous Quantitative Interpretation (QI) and seismic facies classification. The project covers the entire end-to-end pipeline, from rock physics synthetic forward modeling and multi-scenario dataset generation to model evaluation on blind test datasets.🚀 Key FeaturesSynthetic Data Engine: Generates seismic datasets using the Shuey 2-Term approximation and Ricker Wavelet convolution.Multi-Task Learning (MTL): A single, unified architecture with two output branches:Regression: Inverts for P-wave velocity ($V_p$), S-wave velocity ($V_s$), and Density ($\rho$).Classification: Identifies and classifies AVO Class I, II, III, and IV anomalies.Real-World Parameterization: Incorporates geological complexities including frequency attenuation, rock compaction, and Gaussian noise injection to simulate realistic field Signal-to-Noise Ratio (SNR) conditions.Ultra-Deep Target Support: Features adjustable parameterization for ultra-deep targets (up to 10,000 m) reflecting high-pressure and maximum compaction environments.🏗️ Model ArchitectureThe model utilizes a sequential approach to capture the stratigraphic depositional context bidirectionally:Input Layer: 3D Tensor structured as (Samples, Time Steps, Angles).Shared Encoder: Bidirectional(LSTM) layers equipped with BatchNormalization and Dropout to extract robust temporal features and prevent overfitting.Task-Specific Heads:Regression Head: Dense layers with linear activation for continuous estimation of elastic parameters.Classification Head: Dense layers with softmax activation to predict the probability distribution across the 4 AVO facies.📂 Directory StructurePlaintext├── data/
